# 「 🌐 Full Stack Engineering Stack 」

> *"A full stack developer doesn't just connect a frontend to a backend — they architect a seamless, end-to-end user experience."*

---

## ⚡ 1. Frontend — React

My React development is built on modern patterns and best practices:
* **Hooks-First:** All components use functional components with React hooks (`useState`, `useEffect`, `useContext`, `useReducer`).
* **State Management:** Context API for lightweight global state; Redux Toolkit for larger, complex applications.
* **Routing:** React Router v6 with protected routes, lazy loading, and nested layouts.
* **Styling:** Vanilla CSS with CSS variables for theming; clean responsive layouts using Flexbox and Grid.

---

## 🛠️ 2. Backend — Node.js & Express.js

* **Express.js:** Minimal, fast, and flexible REST API framework for building server-side logic.
* **Middleware Stack:** Body parsing, CORS configuration, JWT verification, and error handling — all modular and reusable.
* **Controller Pattern:** Business logic separated from route definitions for clarity and testability.

---

## 🗄️ 3. Database — MongoDB

* **Mongoose ODM:** Schema-based data modeling with type validation, virtual fields, and pre/post hooks.
* **Schema Design:** Embedding vs. referencing trade-offs based on data access patterns and query efficiency.
* **Indexing:** Strategic index creation on frequently queried fields to optimize read performance.
